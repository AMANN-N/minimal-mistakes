## Overview
Implementation of Linear Support Vector Machine from scratch, finding the optimal hyperplane that maximizes the margin between classes.

## Key Components
- Hinge loss function
- Gradient descent optimization
- Margin maximization
- Regularization (L2)
- Binary classification

- **Applications:** Text classification, Image classification, Anomaly detection
