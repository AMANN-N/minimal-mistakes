## Overview
Implementation of K-Means clustering algorithm from scratch for grouping data points into K clusters based on similarity.

## Key Components
- Centroid initialization
- Distance calculation (Euclidean)
- Cluster assignment
- Centroid update
- Convergence detection

## Technologies Used
- **Framework:** NumPy, Matplotlib
- **Concepts:** Unsupervised learning, Clustering, Distance metrics
- **Applications:** Customer segmentation, Image compression, Anomaly detection
