## Overview
Complete implementation of Logistic Regression from scratch using gradient descent for binary classification problems.

## Key Components
- Sigmoid activation function
- Binary cross-entropy loss
- Gradient descent optimization
- Decision boundary
- Probability predictions

## Technologies Used
- **Framework:** NumPy, Matplotlib
- **Concepts:** Binary classification, Maximum likelihood, Gradient descent
- **Applications:** Spam detection, Medical diagnosis, Credit scoring
