## Overview
Implementation of ResNet (Residual Network) architecture that uses skip connections to enable training of very deep neural networks.

## Key Components
- Residual blocks
- Skip connections (identity mapping)
- Batch normalization
- Bottleneck architecture
- Deep network training

## Technologies Used
- **Framework:** PyTorch
- **Concepts:** Residual learning, Skip connections, Deep networks
- **Applications:** Image classification, Object detection, Feature extraction
