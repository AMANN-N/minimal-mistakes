## Overview
Developed a smart Boot + Maven parking system with reservation, billing, and OTP-based email login features.

## Key Features
- Implemented a MySQL backend and deployed securely on Apache Tomcat
- Supported production-like testing and scalability
- OTP-based authentication system
- Real-time reservation and billing

## Technologies Used
- **Languages:** Java
- **Frameworks:** Spring Boot, Maven
- **Database:** MySQL
- **Server:** Apache Tomcat
