## Overview

College Twitter is a lightweight Android application built for **campus-level communication**, allowing students to post, browse, and engage with news and updates within a closed academic community.

The project was built end-to-end, covering UI design, backend integration, and real-time data handling.

---

## Key Details

- Implemented **user authentication and registration** using Firebase Authentication
- Built a **news posting and feed system** backed by Firebase Realtime Database
- Enabled real-time updates so new posts appear instantly across devices
- Focused on simple, intuitive UI for easy campus adoption
- Designed to support basic social interactions in a private academic setting

---

## Tech Stack

- **Platform:** Android
- **Languages:** Java, XML
- **Backend:** Firebase Authentication, Firebase Realtime Database
- **Tools:** Android Studio
