## Overview

This project focuses on **continuous user authentication using facial recognition**, aimed at improving security for cloud-based and real-time systems.

Instead of one-time login verification, the system continuously validates user identity by combining **face detection, recognition, and liveness checks**, while keeping computational overhead low.

---

## Key Details

- Designed a **modular authentication pipeline** where processing stages are triggered only when needed.
- Used **frame differencing** to detect user presence and enable liveness verification.
- Integrated face detection and recognition modules using CNN-based models.
- Optimized the pipeline to **reduce computational load** while maintaining real-time performance.
- Achieved a **25% reduction in false acceptance rate** compared to baseline approaches.
- Balanced accuracy and latency for practical deployment scenarios.
