## Overview
Designed a continuous authentication system using frame differencing, face detection, and liveness checks to reduce false accepts by 25% while maintaining low latency.

## Key Features
- Implemented frame differencing for liveness detection
- Integrated cloud-based storage and analysis
- Balanced latency vs accuracy for real-time deployment
- Reduced false acceptance rate by 25%

## Technologies Used
- **Languages:** Python
- **Libraries:** OpenCV, TensorFlow, PyTorch
- **Concepts:** Computer Vision, Biometric Authentication, Cloud Integration
