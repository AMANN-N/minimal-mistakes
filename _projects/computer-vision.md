## Overview
Developed a real-time computer vision application for object detection, tracking, and analytics in video streams.

## Key Features
*   Multi-object tracking with 95% accuracy
*   Custom-trained models for specific use cases
*   Real-time dashboard for analytics

## Technologies Used
*   PyTorch, YOLOv8, DeepSORT
*   OpenCV, CUDA
*   React, WebSockets
