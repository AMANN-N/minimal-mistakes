## Overview
Built a goal-oriented learning framework for simulated conversations, combining SEQ2SEQ models with RL to deliver more context-aware responses.

## Key Features
- Implemented a bi-directional RNN with global attention
- Developed full training/evaluation pipelines for iterative performance improvement
- Applied reinforcement learning techniques to improve conversation quality

## Technologies Used
- **Languages:** Python
- **Frameworks:** PyTorch, TensorFlow
- **Concepts:** Sequence-to-Sequence models, Reinforcement Learning, NLP
